Frequently Asked

Is this a replacement for my cloud SIEM?

Sentinal can replace or augment cloud SIEMs. Many users use us as a first-line filter to reduce cloud ingestion costs by 80%.

How do you handle data privacy?

Your data never leaves your network. All analysis, ML training, and log storage happens on your own disk and memory.

Can it run on Docker?

Yes, our Core node is optimized for Docker and Kubernetes. Agents are available as binaries or containers for maximum flexibility.

Does it support auto-response?

Absolutely. You can define playbooks that trigger local firewall changes or isolate containers the millisecond a threat is confirmed.
